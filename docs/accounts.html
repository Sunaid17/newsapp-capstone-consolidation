accounts module

Authentication views for user registration, login, and logout.

Supports registration with role selection: - Reader: Can browse and bookmark articles - Journalist: Can create and manage articles - Editor: Can review and approve articles

accounts.views.profile(request)[source]

Display user profile information.

accounts.views.register(request)[source]

Handle user registration with role selection.

Allows users to register as: - ‘reader’: Can browse and bookmark articles - ‘journalist’: Can create and submit articles for review - ‘editor’: Can review and approve articles

accounts.views.user_login(request)[source]

Handle user authentication and login.

Authenticates user and logs them in with their appropriate role.

accounts.views.user_logout(request)[source]

Handle user logout and redirect to home page.
