articles module

Models for the News Application.

This module defines the database models for users, articles, newsletters, and categories. Includes CustomUser with role-based access control.

Supports four roles: - ‘reader’: Can browse and bookmark articles, subscribe to publishers - ‘journalist’: Can create and manage their own articles and newsletters - ‘editor’: Can review, approve, and manage all articles - ‘admin’: Full system access
